# mpic++

> Open MPI wrapper compiler for C++.
> See also: `mpirun`.
> More information: <https://manned.org/mpicxx>.

- Compile an Open MPI program:

`mpic++ {{path/to/source_file}}`

- Show all the wrapper-supplied flags:

`mpic++ --showme`
